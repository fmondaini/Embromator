Embromator - Pseudo 'Lorem Ipsum' Generator
===========================================


### Dependencies:
- Django 1.4
- Python 2.7

### Inspired by:
 - LeroLero.com

### How to install:
Installation:
> <code> $ pip install -r requirements.txt </code>

Run:
> <code> $ python manage.py runserver </code>
